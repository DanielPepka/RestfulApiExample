RestfulApiExample.Web is used to demonstrate examples of using entity framework to read and write from a sql database using restful webapi endpoints. 

The project also heavily utilizes
* KnockoutJS: https://knockoutjs.com/
* Typescript: https://github.com/Microsoft/TypeScript

The database tables are created using a Code First Migration style utilizing entity framework which is all stored in the DataAccess library.
The database tables themselves are represented by POCO objects stored in the Data library.
